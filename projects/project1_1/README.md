## 🎯 项目概述

本项目是一个基于 LangChain 框架构建的多任务智能问答助手，能够：

- **自然语言对话**：理解用户意图，进行智能对话
- **天气查询**：通过高德地图 API 查询城市天气信息
- **信息搜索**：通过 Tavily Search API 搜索最新新闻和信息
- **对话历史**：维护上下文，支持多轮对话
- **工具自动调用**：LLM 自动判断何时调用工具

### 核心特性

- ✅ 基于 LangChain LCEL 构建的对话链
- ✅ 支持 Function Calling（工具自动调用）
- ✅ 使用 DeepSeek API 作为底层 LLM（兼容 OpenAI 格式）
- ✅ 完整的错误处理和日志记录
- ✅ 配置管理和环境变量支持
- ✅ 结构化日志系统

## 🚀 核心功能

### 1. 自然语言对话

支持日常对话，理解用户意图，提供智能回复。

**示例：**
```
用户: 你好，AI 助手，你能为我做什么？
助手: 你好！我是一个多任务问答助手，可以帮你：
- 查询天气信息
- 搜索最新新闻和信息
- 进行日常对话
```

### 2. 天气查询

自动识别天气查询意图，调用高德地图 API 获取实时天气信息。

**示例：**
```
用户: 查询北京天气
助手: 根据高德地图API，北京当前天气情况如下：
- 温度: 15°C
- 天气: 晴
- 风向: 南风
- 风力: 3级
```

### 3. 信息搜索

自动识别搜索意图，调用 Tavily Search API 获取最新信息。

**示例：**
```
用户: 搜索最新AI技术发展
助手: 根据搜索结果，以下是关于AI技术发展的最新信息：
[显示搜索结果摘要和链接]
```

## 🏗️ 技术架构

### 技术栈

- **LangChain**: 对话链构建和工具集成
- **DeepSeek API**: 大语言模型（兼容 OpenAI API）
- **Pydantic**: 数据验证和类型检查
- **Loguru**: 结构化日志记录
- **Python-dotenv**: 环境变量管理

### 模块说明

#### 1. agents/qa_agent.py
- **功能**: 问答代理核心逻辑
- **职责**:
  - 管理对话历史
  - 处理用户输入
  - 调用工具（如果需要）
  - 生成回复
- **设计模式**: LCEL 链式调用、工具绑定

#### 2. tools/
- **amap_weather_tool.py**: 高德地图天气查询工具
- **tavily_search_tool.py**: Tavily 搜索工具
- **tool_schemas.py**: 工具参数验证模式（Pydantic）

#### 3. config/settings.py
- **功能**: 统一配置管理
- **特性**:
  - 使用 Pydantic BaseSettings
  - 支持环境变量自动加载
  - 配置验证和类型检查

#### 4. core/logger.py
- **功能**: 结构化日志系统
- **特性**:
  - 多级别日志（DEBUG, INFO, WARNING, ERROR）
  - 文件和控制台输出
  - 日志轮转和压缩

## 🚀 快速开始

### 1. 环境要求

- Python 3.11+
- pip 或 conda

### 2. 安装依赖

```bash
# 克隆或进入项目目录
cd projects/project1_1

# 创建虚拟环境（推荐）
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

### 3. 配置环境变量

创建 `.env` 文件（在项目根目录）：

```bash
# DeepSeek API 配置（必需）
DEEPSEEK_API_KEY=your_deepseek_api_key_here
DEEPSEEK_API_BASE=https://api.deepseek.com/v1

# 高德地图 API 配置（必需）
AMAP_API_KEY=your_amap_api_key_here

# Tavily 搜索 API 配置（必需）
TAVILY_API_KEY=your_tavily_api_key_here

# 应用配置（可选）
LOG_LEVEL=INFO
MAX_CONVERSATION_HISTORY=50
```

### 4. 运行程序

```bash
python main.py
```

## ⚙️ 配置说明

### API 密钥获取

#### 1. DeepSeek API
- 访问: https://platform.deepseek.com/
- 注册账号并获取 API Key
- 设置环境变量: `DEEPSEEK_API_KEY`

#### 2. 高德地图 API
- 访问: https://lbs.amap.com/
- 注册开发者账号
- 创建应用并获取 Key
- 设置环境变量: `AMAP_API_KEY`

#### 3. Tavily Search API
- 访问: https://tavily.com/
- 注册账号并获取 API Key
- 设置环境变量: `TAVILY_API_KEY`

### 配置验证

运行环境设置脚本验证配置：

```bash
python scripts/setup_environment.py
```

## 💻 开发指南

### 添加新工具

1. 在 `tools/` 目录创建工具类
2. 在 `tools/tool_schemas.py` 定义参数模式
3. 在 `agents/qa_agent.py` 中注册工具

### 扩展功能

- **缓存**: 可以集成 Redis 缓存工具调用结果
- **多模态**: 可以添加图片、语音处理功能
- **持久化**: 可以添加数据库存储对话历史
