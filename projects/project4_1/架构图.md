## 状态流转架构

```mermaid
stateDiagram-v2
    [*] --> START
    START --> input_proc : 用户输入
    input_proc --> chatbot : 处理完成

    chatbot --> tools : 需要调用工具
    chatbot --> END : 无需调用工具

    tools --> chatbot : 工具执行完成

    END --> [*]
```

## 数据流架构

```mermaid
sequenceDiagram
    participant User as 用户
    participant Input as 输入处理
    participant LangGraph as LangGraph工作流
    participant LLM as 大语言模型
    participant Tools as 工具
    participant DB as 数据库
    participant RAG as RAG检索

    User->>Input: 发送输入(文本/音频/图片)
    Input->>LangGraph: 处理后文本
    LangGraph->>LLM: 发送对话历史
    alt 需要工具
        LLM-->>LangGraph: 返回工具调用
        LangGraph->>Tools: 执行工具
        alt 订单查询
            Tools->>DB: 查询订单
            DB-->>Tools: 返回订单数据
        else 政策检索
            Tools->>RAG: 检索政策
            RAG-->>Tools: 返回相关文档
        end
        Tools-->>LangGraph: 返回工具结果
        LangGraph->>LLM: 发送工具结果
    end
    LLM-->>LangGraph: 返回最终响应
    LangGraph-->>User: 显示客服响应
```
