# 项目流程图

## 系统初始化流程

```mermaid
flowchart TD
    A[程序启动] --> B[setup_database<br/>初始化数据库]
    B --> C{数据库是否为空?}
    C -->|是| D[插入示例订单数据]
    C -->|否| E[跳过数据插入]
    D --> F[setup_rag_retriever<br/>初始化RAG检索器]
    E --> F
    F --> G[初始化LLM<br/>真实模型或Mock]
    G --> H[绑定工具到LLM]
    H --> I[构建LangGraph工作流]
    I --> J[配置SqliteSaver<br/>持久化检查点]
    J --> K[编译工作流为app]
    K --> L[等待用户输入]
```

## 对话查询流程

```mermaid
flowchart TD
    A[用户输入] --> B{输入类型?}
    B -->|文本| C[直接处理]
    B -->|音频| D[process_audio_input<br/>ASR转写]
    B -->|图片| E[process_image_input<br/>OCR识别]
    D --> C
    E --> C
    C --> F[添加HumanMessage到状态]
    F --> G[input_proc节点<br/>输入预处理]
    G --> H[chatbot节点<br/>LLM处理]
    H --> I{是否需要工具调用?}
    I -->|是| J[tools节点<br/>执行工具]
    I -->|否| K[返回AI响应]
    J --> L{工具类型?}
    L -->|check_order| M[查询orders.db]
    L -->|search_policy| N[RAG检索知识库]
    M --> O[返回工具结果]
    N --> O
    O --> P[添加ToolMessage到状态]
    P --> H
    K --> Q[保存检查点到checkpoints.db]
    Q --> R[显示客服响应]
```

## 工具调用流程

```mermaid
flowchart TD
    A[LLM生成tool_calls] --> B[解析工具名称和参数]
    B --> C{工具名称?}
    C -->|check_order| D[调用check_order工具]
    C -->|search_policy| E[调用search_policy工具]

    D --> F{参数order_id?}
    F -->|有效| G[连接orders.db]
    F -->|无效| H[返回错误信息]
    G --> I[执行SQL查询]
    I --> J{查询结果?}
    J -->|找到| K[格式化订单信息]
    J -->|未找到| L[返回未找到提示]
    K --> M[返回工具结果]
    L --> M
    H --> M

    E --> N[初始化RAG检索器]
    N --> O[使用query检索文档]
    O --> P[获取相关文档列表]
    P --> Q[拼接文档内容]
    Q --> R[返回拼接结果]

    M --> S[生成ToolMessage]
    R --> S
    S --> T[添加ToolMessage到对话历史]
    T --> U[调用LLM生成最终响应]
```

## 多轮对话流程

```mermaid
flowchart TD
    A[第一轮对话] --> B[创建thread_id<br/>session_1]
    B --> C[发送用户消息1]
    C --> D[处理并保存检查点1]
    D --> E[返回AI响应1]
    E --> F[第二轮对话]
    F --> G[使用相同thread_id]
    G --> H[加载检查点1<br/>恢复对话历史]
    H --> I[发送用户消息2]
    I --> J[处理并保存检查点2]
    J --> K[返回AI响应2]
    K --> L[第N轮对话]
    L --> M[继续使用相同thread_id]
    M --> N[加载最新检查点<br/>恢复完整对话历史]
    N --> O[处理新消息]
    O --> P[保存新检查点]
    P --> Q[返回AI响应N]
```

## RAG检索流程

```mermaid
flowchart TD
    A[用户查询政策] --> B[调用search_policy工具]
    B --> C[setup_rag_retriever<br/>初始化检索器]
    C --> D{Embeddings类型?}
    D -->|生产环境| E[OpenAIEmbeddings<br/>/DashScopeEmbeddings]
    D -->|演示环境| F[FakeEmbeddings]
    E --> G[创建InMemoryVectorStore]
    F --> G
    G --> H[从知识库加载文档]
    H --> I[将文档转换为向量]
    I --> J[存储到VectorStore]
    J --> K[retriever.invoke(query)]
    K --> L[计算查询向量]
    L --> M[搜索相似文档]
    M --> N[返回Top-K文档]
    N --> O[拼接文档内容]
    O --> P[返回给LLM]
    P --> Q[LLM基于检索内容生成回答]
```

## 数据库操作流程

```mermaid
flowchart TD
    A[需要数据库操作] --> B{操作类型?}
    B -->|初始化| C[setup_database]
    B -->|查询订单| D[check_order工具]
    B -->|保存检查点| E[SqliteSaver]

    C --> F[连接orders.db]
    F --> G[CREATE TABLE<br/>IF NOT EXISTS]
    G --> H{表是否为空?}
    H -->|是| I[插入示例数据]
    H -->|否| J[关闭连接]
    I --> J

    D --> K[连接orders.db]
    K --> L[参数化SQL查询]
    L --> M[fetchone获取结果]
    M --> N[关闭连接]
    N --> O[格式化返回值]

    E --> P[连接checkpoints.db]
    P --> Q[自动创建检查点表]
    Q --> R[保存当前状态]
    R --> S[提交事务]
    S --> T[关闭连接]
```
